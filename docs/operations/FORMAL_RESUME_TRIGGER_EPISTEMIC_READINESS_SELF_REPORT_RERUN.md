## Explicit Non-Authorization

This trigger does not authorize:

- edits under `core/`, `cli/`, `extensions/`, `.cerebro/`, or root `tests/`;
- edits to `core/schema.py` or any canonical state file;
- creating a runtime gate, canonical claim graph, or permission layer;
- treating advisory readiness as approval to act;
- third-party project mutation;
- changing extraction/evaluation behavior unless the existing generator fails
  its focused tests, in which case stop and open a separate trigger.

## Stop Conditions

Stop immediately if:

- the AGENTS-equivalent gate is red before edits;
- the manifest needs to read `.cerebro/` or escape the repository root;
- the generated report requires unbounded reads or more than 24 sources;
- any generated finding is interpreted as canonical truth or permission;
- focused `epistemic_readiness` tests fail;
- architecture or doc-governance tests fail;
- the final AGENTS-equivalent gate is red.
